Extension Cord Help

Getting Started

Thank you for using Extension Cord. It looks like you've successfully installed Extension Cord and have it running, so this document will focus on usage of the application itself.

For help installing or configuring Extension Cord, please see the installation documentation.

Test Cases and Test Plans

Extension Cord's basic units of measuree are test cases and test plans.

  • A test case is a discrete, verifiable behaviour, which may include one or more steps.
  • A test plan is a collection of test cases, to be performed over a planned period of time.

A test case may be added to one or more test plans, and test plans may contain many test cases. Test cases are organized in a hierarchical tree of folders.

Creating Test Cases

We'll begin by adding a test case to Extension Cord. This set of steps assumes that this is your first test case, and includes instructions on adding a folder.

  1. Click "Test Cases" from the top navigation.
  2. Click the "+folder" button from beneath the folder navigation on the left side of the page. You will be prompted to choose a name for your folder; enter a name in the input box and click "save".
  3. Click the folder you just added, then click the "+Test Case" button, which appears at the top of the page.
  4. Complete the form, and click "save".
  5. Your test case is now saved. You may navigate to your folder and review it.

Creating Test Plans

Now that you have a test case, you can add it to a test plan by following these steps:

  1. Click "Test Plans" from the top navigation.
  2. Click the "+Test Plan" button
  3. Complete the form, and click "save".
  4. Your test plan is now saved, and you will be directed to a page to add test cases to your new test plan.
  5. Navigate to the folder that contains the test cases you would like to add to your test plan, or use the search dialog to find them.
  6. Select the test cases you would like to add from the list, and click "Add Test Cases".
  7. When you're finished adding test cases, click the "progress report" link. This will show you a summary of your test plan and progress report.

Tracking Your Results

Now you're ready to execute your test plans. That's why we're here, right?

Here's what to do:

  1. Find your test plan. The Extension Cord homepage lists "active" test plans, where active is defined as testpans that have a start date that is in the past and an end date that is in the future.
  2. Click on your test plan, then click on "Add Results".
  3. Find the test cases you have executed. This can be done by navigating the folder tree, or by using the search form.
  4. Select the test case(s) you would like to update from the list, then click one of the status buttons at the top of the list ("passed", "failed", "blocked", etc)
  5. Navigate back to the Progress Report page, the results you've added will be reflected on the chart at the top of the page and the chart in the middle of the page.

You can change the status of a test after it's been run an unlimited number of times. Each change will be recorded in the test case's execution history.

API Access

Extension Cord features a REST api that can be used for integration with other tools and advanced reporting, among other things. The REST api has it's own documentation that can be found here.
